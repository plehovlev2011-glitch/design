Генератор кода для кликабельных областей

Ваше изображение: 1920x1080 пикселей

Координаты областей (примерные):

1. about: X=26, Y=316 (1.354%, 29.259%)
2. contacts: X=245, Y=315 (12.760%, 29.167%)
3. cases: X=498, Y=316 (25.938%, 29.259%)
4. whyme: X=664, Y=316 (34.583%, 29.259%)
5. more: X=873, Y=317 (45.469%, 29.352%)
6. rightsnlaws: X=16, Y=595 (0.833%, 55.093%)
7. designphilosophy: X=14, Y=25 (0.729%, 2.315%)

Проблема: Все области имеют ширину и высоту 0px.

Решение: Нужно указать правильные размеры для каждой области.

Как настроить размеры:

  1. Откройте ваш файл hello.png в любом графическом редакторе
  2. Измерьте размер каждой кнопки в пикселях (ширина и высота)
  3. Обновите значения w (ширина) и h (высота) в массиве areas в коде выше
  4. Еще раз нажмите "Скопировать исправленный код"

Пример правильной области:

{name: "about", x: 26, y: 316, w: 150, h: 60, link: "about.html"}
// где w=150px (ширина кнопки), h=60px (высота кнопки)
